# 🤝 المساهمة في Genetic Cipher

## كيف تساهم؟

1. Fork المستودع
2. أنشئ branch جديد: `git checkout -b feature/اسم-الميزة`
3. اcommit التغييرات: `git commit -m "✨ إضافة ميزة جديدة"`
4. ارفع الbranch: `git push origin feature/اسم-الميزة`
5. افتح Pull Request

## قواعد الكود

- استخدم ESLint
- اكتب tests للميزات الجديدة
- حافظ على التوثيق محدث

## التواصل

- Issues للأخطاء
- Discussions للأفكار
