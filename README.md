## 🚀 التشغيل السريع

```bash
# 1. Clone
 git clone https://github.com/AaydFaisalAbdoMohammed/GeneticCipher.git
 cd GeneticCipher

# 2. Install
 npm install

# 3. Run
 npx expo start
 # Press 'a' for Android
 # Press 'i' for iOS
```

## 📦 النشر

### Google Play Store
```bash
npx eas build --platform android --profile production
npx eas submit --platform android
```

### Apple App Store
```bash
npx eas build --platform ios --profile production
```
